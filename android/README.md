# Vexta Android Uygulaması

Bu klasör Vexta Android uygulamasının kaynak kodunu barındıracaktır.

## Teknoloji
- Kotlin
- Android Studio
- Firebase (Auth, Firestore, Storage)

## Kurulum (yakında)
1. Android Studio kurulacak  
2. Proje oluşturulacak  
3. Firebase entegrasyonu yapılacak  
4. İlk ekran (Login) hazırlanacak  

## Modüller (plan)
- Auth (Giriş / Kayıt)
- Feed (Dikey dizi listesi)
- Player (Dikey video oynatıcı)
- Profil (Jeton / VIP)
